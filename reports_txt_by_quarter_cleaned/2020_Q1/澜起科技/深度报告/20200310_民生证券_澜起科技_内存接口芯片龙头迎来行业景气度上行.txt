● 服务器行业复苏, 内存接口芯片迎来增长引擎
内存接口芯片主要用于服务器领域。一台服务器根据高中低端定位不同、搭载
1-4 颗 CPU 芯片, 一颗 CPU 通常搭配一定数量的内存接口芯片, 因此内存接口
芯片景气度与服务器出货量高度相关。我们认为受益云计算需求拉动,服务器出
已明显回升。公司作为内存接口芯片行业全球第二大厂商,将深度受益行业发展。
此外,公司与全球服务器 CPU 龙头 Intel 合作多年, Intel 占全球服务器 CPU 市
场份额超过 90%, 且 Intel 间接持有公司 9%股份,与 Intel 多层次深度绑定将助
力公司发展。

● DDR5技术步入市场, 搭載更多内存接口芯片带来发展机遇
新一代内存技术 DDR5 引进了一系列可以增强内存模组可靠性、可用性和可维护
产品, DDR5 的市场采用率有望迅速提升, 预计 2021 年 LPDDR5+DDR5 产品的
市场采用率合计可达 45%。支持 DDR5 协议的内存模组将搭载更多的内存接口
芯片,由 DDR4的10个芯片上升到11个芯片。此外,DDR5将增加PMIC(电
源管理芯片)、TS(温度传感器)等组件,进一步带动内存接口芯片价值量提升。

全球内存接口芯片巨头,深耕行业十余载
公司深耕内存接口芯片领域十余年,目前已是全球第二大供应商。公司成立于2004年,
2006 年开始针对云计算市场的存储器接口提供解决方案,起初主要为 Intel 供应 FBDIMM(全
缓冲双列直插内存模组）产品。2007-2009 年期间，公司接到大客户三星的第一个 AMB（高
级内存缓冲器）订单，开始为全球最大互联网公司的数据中心提供 DDR3（第三代双倍速率
内存技术标准)寄存缓冲芯片。2011年,公司收购摩托罗拉杭州中心,营收首次超过5000
万美元并实现盈利，产品重心开始转移至 LRDIMM（减载双列直插内存模组）。2013 年，公
司开始为 RDIMM (寄存式双列直插内存模组) 的缓冲寄存器和 LRDIMM 的存储缓冲器提供
解决方案。2014 年，公司推出支持 Intel Xeon（至强）ES-2600 V3 系列处理器的内存接口套
片, 紧接着公司发明的 DDR4 全缓冲“1+9”架构被采纳为国际标准, 奠定了公司的行业领
先地位,目前公司已是全球内存接口芯片的第二大供货商。2016年后,公司开始与清华大学、
Intel 联合研发津速®服务器 CPU, 2018 年成功推出第一代津速®服务器平台产品, 目前处于
市场推广阶段，有望成为公司下一个业务增长点。

2.2 内存接口芯片增长率大于服务器增速,公司将显著受益
内存接口芯片是 CPU 与内存 (DRAM) 之间进行数据传输的核心部件，合格的供货商
需要获得服务器产业链的三轮严格认证。内存接口芯片属于内存模组的一部分,主要应用于
服务器领域,帮助服务器 CPU 存取内存中的数据,通过内存接口芯片对数据进行存储和缓冲,
可以提升 CPU 对内存数据访问的速度和稳定性,满足服务器 CPU 对内存模组日益增长的容
量和性能需求。内存接口芯片需要与内存厂商生产的各种内存颗粒和内存模组进行配套，同
时需要通过 CPU 厂商 (Intel 等)、内存供应商(三星、SK 海力士等) 和服务器厂商在稳定
性、运行速度及功耗等方面的三轮严格认证，其中以 CPU 厂商的认证最为关键。公司支持
DDR2-DDR4 协议标准的内存接口芯片已经先后获得行业认证，目前是全球唯一在 DDR3 和
DDR4 两代内存全缓冲解决方案得到 Intel 认证的厂商, 同时正在积极极研发符合 DDR5 标准
的寄存时钟驱动器芯片及数据缓冲器芯片。(注: DDR (Double Data Rate) 是内存模块中使
输出增加一倍的技术, 已经成为目前主流的内存规范, 受到各大芯片组厂商主流产品的支持。)
增速。内存接口芯片是内存模组的核心器件, 单个 CPU 一般需配备多个内存模组，随着服
务器出货量的增长，对内存接口芯片的需求也将成倍增长。此外，随着服务器数据处理能力
的提升，单台服务器搭载的 CPU 核心数量也在增长，一般低端服务器需要搭载 1 个 CPU 核
心,中端服务器要搭载 2-3 个 CPU 核心,高端服务器需要搭载 4 个 CPU 核心,从而进一步
拉动对内存接口芯片的需求。

DDR5 技术步入市场, 搭载更多内存接口芯片带来发展机
公司内存接口芯片产品覆盖 DDR2-DDR4 协议标准，发明 DDR4 全缓冲“1+9”架构奠定
行业领先地位。内存接口芯片按照功能可以分为寄存缓冲器（RCD）、数据缓冲器（DB）和
内存缓冲器(MB)三类。RCD用于存储缓冲来自内存控制器的地址/命令/控制信号，作用是
进行数据传输的控制和分配, DB 用于存储来自内存控制器或内存的数据信号, MB 同时具备
RCD 和 DB 功能, 可由单颗芯片(高级内存缓冲器 AMB 或 MB)实现, 也可由 RCD 和 DB
套片实现，搭载不同数量的 RCD、DB 和 MB 构成了不同内存接口协议（DDR）标准下的
DRAM 模组。此前, DDR2 和 DDR3 标准下的内存接口芯片,均是采用集中式的单颗 MB 或
者 RCD 芯片，到 DDR4 时代，公司创新式地发明 DDR4 全缓冲“1+9”架构，利用 1 颗 RCD +
9 颗 DB 组成一套分布式的内存接口芯片，大幅减少了 CPU 与内存颗粒间的负载，降低了信
号传输损耗, 有效解决了内存存储的高容量与 CPU 处理的高速度之间的矛盾, 之后被 JEDEC
(全球微电子产业的领导标准机构) 采纳为国际标准, 从而奠定了公司在内存接口芯片领域
的领先地位。

内存市场迎来 DDR5 技术升级, 公司积极参与相关内存接口芯片的标准制定, 以保持技
术领先优势。随着云计算、大数据的兴起，服务器的数据容量和处理速度也在不断提高，间
接推动了 DDR 技术的升级迭代。相比 DDR4, DDR5 主要在功耗、容量和传输速率上进行了
改进: 1) 功耗方面,从 DDR4 支持的 1.2V 降低到 1.1V,进一步降低了功耗; 2) 容量上面,
DDR5 内存将从 8GB 起步,单个内存条最高可达 32GB,是 DDR4 存储容量的 2倍:3)传
输速率方面，DDR5 可以轻松实现 4800MT/s 的高运行速率，最高运行速率可达 6400MT/s,
是 DDR4 的 2 倍。此外，DDR5 的带宽从 25.6GB/s 提升到 32GB/s，预取位数从 8 位增加到
16 位, 大幅提升了数据传输性能。目前 DDR4 已经处于成熟期, DDR5 引进了一系列可以增
强内存模组 RAS（可靠性、可用性和可维护性）性能的新功能,预计未来几年将逐步取代
DDR4。
图 24: 公司内存接口芯片产品随着 DDR 技术的升级而不断更新迭代
随着 DDR 相关技术的升级，内存模组的容量和价值也在不断上升，从而推动内存接口
芯片的数量和价值增加。DDR4时代,公司推出的LRDIMM使用的内存接口芯片数量由DDR3
的 1 个增加到 10 个 (9 个 DB + 1 个 RCD), 到 DDR5 时代, 内存接口芯片数量将继续增加
到 11 个 (10 个 DB + 1 个 RCD), 单个内存模组的内存接口芯片价值也将随之增加。
图 25: DDR5 内存接口芯片采用“1+10”架构
此外, DDR5 将增加 PMIC (电源管理芯片)、TS (温度传感器) 等组件, 进一步带动
内存接口芯片价值量提升。1)为了降低功耗, DDR5 的工作电压从 DDR4 的 1.2V 降到 1.1V,
信号的噪声容限变得更小，需要 PMIC 更有效地控制电源负载，同时为了满足 DDR5 内存颗
粒高容量和高速度的需要，PMIC 将从主板（motherboard）转移到每个内存模块上，从而大
幅提高散热效率和减少电压波动，实现电源模块更小尺寸、更加节能,进而拉升 PMIC 的搭
载价值; 2) 为了实现优异的温度控制, DDR5 将搭载更多的 TS 传感器(从 DDR4 的 1 个增
加到 2 个); 3) 随着组件数量和信号速度的提升, DDR5 将采用更加节省空间和高效率的系
统管理总线解决方案,同时实现更高的可靠性和可扩展性。
未来随着 DD5 相关产品的引入, DDR5 的市占率将迅速提升。据 Cadence analysis 预测,
2020-2022 年, DDR5 市占率将分别达到 7%、16%和 25%, CAGR 可达 89%, LPDDR5 市占
率将分别达到 12%、29%和 35%, CAGR 可达 71%。
全球三大厂商备战 DDR5 内存接口芯片,公司在研发上处于领先地位,有望于 2021 年
实现相关产品量产。全球目前主要有三家公司积极研发符合 DDR5 标准的内存接口芯片，分
别是澜起科技、IDT 和 Rambus。Rambus 于 2017 年宣布研发 DDR5 内存接口芯片,目前已
具备生产工程样片的能力, IDT 于 2018 年宣布研发出首款可用于 DDR5 内存接口芯片组的
寄存时钟驱动芯片(RCD)。公司第一阶段研发的 DDR5 产品包括 Gen 1.0 DDR5 寄存时钟驱
动器芯片和 Gen 1.0 DDR5 数据缓冲器，有望在 2021 年实现第一代 DDR5 芯片的量产。

公司积极布局津速®服务器 CPU、人工智能(AI)处理器芯片和 SOC 芯片, 短期创收
万元, 32%用于津速®服务器 CPU 及其平台技术升级项目, 23%用于 AI 芯片研发项目。随着
国内云计算、大数据平台的兴起，加上国内对核心硬件技术自主可控的需求愈发强烈，公司
的津逮服务器 CPU 和 AI 芯片业务虽短期创收有限,但未来成长空间巨大。

服务器出货量不及预期的风险:内存接口芯片是内存模组的核心逻辑器件,服务器需要
搭载多个内存模组，终端客户对服务器的采购直接决定了对内存接口芯片的需求，如果服务
